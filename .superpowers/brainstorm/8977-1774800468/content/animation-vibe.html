What animation personality should PingCRM have?

This sets the tone for all microinteractions — speed, easing, intensity

A

Subtle & Professional

Quick, barely-there transitions. 150-200ms durations. Linear/ease-out easing. Think: Notion, Linear. The UI feels responsive but animations never draw attention to themselves. No bouncy effects.

Examples: Fade-in 150ms, color transitions 200ms, no scale/translate on hover, dropdowns appear with subtle opacity fade
B

Polished & Energetic

Smooth spring-like animations. 250-400ms durations. Ease-out curves. Think: Vercel, Stripe dashboard. Cards lift on hover, elements slide into view, interactions have satisfying feedback. Noticeable but not distracting.

Examples: Cards lift 2px + shadow on hover, staggered fade-up on page load, dropdowns scale from origin, send button pulses briefly on success
C

Playful & Delightful

Bouncy spring physics, overshoots, particle effects. 300-500ms durations. Spring easing. Think: Figma, Raycast. Interactions feel fun — confetti on success, elements bounce into place, hover effects are dramatic.

Examples: Cards scale up 1.03x on hover with spring, checkmark draws on success, counters animate when values change, emoji reactions bounce in
